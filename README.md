# REO — Offline Voice Assistant


1. **Install Ollama** — https://ollama.com/download then pull a model:
 
   ```bash
   ollama pull llama3.2:3b
   ```
   Make sure Ollama is running (it lives in your system tray after install).
  if you dont have bash you can use cmd/terminal
---

## 2. Double-click Reo.exe — done!
